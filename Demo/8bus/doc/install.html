8 Substation Model

By default, the CyPSA application runs on an 8 substation model. The model lives at the CPTL Power GitHub as an example of CPTL. Therefore, one must download it as a dataset for the CyPSA application. In the future, this can be turned into a script.

  1. cd projects/8bus
  2. mkdir cptl
  3. git init
  4. git remote add -f origin https://github.com/ITI/cptl-power.git
  5. git config core.sparsecheckout true
  6. echo data/substation_network >> .git/info/sparse-checkout
  7. echo data/substation_yard >> .git/info/sparse-checkout
  8. git checkout master

Concordion

Concordion provides a way to relate a specification with a set of unit tests. Since the project is being developed in Python, we do the following:

  • Download PyConcordion.
  • Download PyConcordion2.
  • Run ant test from the command line to run the CyPSA tests.
  • Look at the output generated in the /tmp/pyConcordion/ folder.
