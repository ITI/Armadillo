CyPSA is a Cyber-Physical Security Assessment toolset designed to improve reliability of grid operations. CyPSA researchers are developing scalable grid modeling, monitoring, and analysis tools to improve grid resiliency to system failures and cyber attacks.

Theoretical and practical techniques are leveraged from cyber security and power engineering domains to develop new algorithms and software tools capable of analyzing real-world threats against power grid critical infrastructures including cyber components (e.g. communication networks), physical components (e.g. power lines), and interdependencies between the two in models and simulations.
